# 📡 Què és DHCP?

DHCP (Dynamic Host Configuration Protocol) és el servei que permet:

- assignar adreces IP automàticament,
- configurar màscara de xarxa,
- definir gateway,
- assignar DNS,
- i gestionar la distribució dinàmica d’adreces.

---

# ⚙️ Importància del servei DHCP

En una xarxa professional, DHCP permet:

- evitar configuracions manuals,
- reduir errors humans,
- centralitzar la gestió de xarxa,
- facilitar l’escalabilitat,
- i automatitzar la connexió dels dispositius.
