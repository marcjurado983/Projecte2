# T09: Donar-se d’alta en un hosting gratuït - Aplicacions Web & PI

## 👤 Tasca individual

En aquesta tasca individual, cada alumne haurà de dur a terme una investigació pròpia per localitzar un servei de hosting gratuït que permeti fer proves.

---

# 🎯 Objectiu de la tasca

L’objectiu principal és:

- experimentar amb un entorn real de servidor web,
- disposar d’un espai de proves,
- i entendre el procés real de desplegament d’una web.

---

# 🔎 Fase 1 — Investigació de hosting gratuït

Cal cercar i seleccionar un proveïdor de:

# 🆓 Hosting gratuït

## 📌 El servei ha de permetre:

- allotjar una web funcional,
- suportar PHP i bases de dades (si és possible),
- accés a panell de control,
- instal·lació de CMS (WordPress o similar),
- ús per a proves educatives.

---

# 🧠 Criteris de selecció

A l’hora d’escollir el servei, cal tenir en compte:

- ⚡ facilitat d’ús
- 💰 cost (ha de ser gratuït)
- 🛠️ funcionalitats disponibles
- 📦 limitacions del pla free
- 🔐 seguretat bàsica
- 🌍 reputació del proveïdor

---

# 🚀 Fase 2 — Creació del compte

Un cop escollit el proveïdor:

- crear un compte personal,
- activar el servei de hosting,
- accedir al panell de control.

---

# 🌐 Fase 3 — Instal·lació de WordPress

## 📌 Objectiu

Instal·lar un sistema de gestió de continguts:

# 🧱 WordPress

---

## ⚙️ Requisits mínims

La instal·lació ha de:

- ser funcional,
- permetre accés a l’administració,
- mostrar una pàgina web activa,
- estar desplegada correctament al hosting.

---

# 🧪 Resultat esperat

Al final de la tasca haureu de tenir:

- 🌐 un hosting gratuït operatiu
- 🧱 WordPress instal·lat i funcionant
- 🔐 accés al panell d’administració
- 📄 evidències del procés realitzat

---

# 🧠 Competències treballades

## 🌐 Web hosting
- serveis d’allotjament
- configuració de servidors web
- entorns de proves

## 🧱 CMS
- instal·lació de WordPress

## 🔎 Investigació tècnica
- selecció de proveïdors
- anàlisi de serveis
- criteri de decisió

## 🧑‍💻 Autonomia professional
- treball individual
- resolució de problemes
